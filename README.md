## Descrizione del Progetto
Per questo test tecnico ho creato un sito web statico utilizzando HTML e CSS in Visual Studio Code.
Il sito è composto da tre sezioni principali: header, body e footer.


## Funzionalità JavaScript
Per aggiungere un comportamento dinamico al sito, ho implementato un event listener sul logo di Sistech: quando l’utente clicca sul logo, la pagina scorre automaticamente verso la sezione “Why Join Sistech”, permettendo di visualizzarne i contenuti.

Questa funzionalità è stata implementata utilizzando:
- Selezione del DOM (getElementById)
- Gestione degli eventi (addEventListener)
- Manipolazione dinamica degli elementi

## Tecnologie Utilizzate
- HTML
- CSS 
- JavaScript
- VSCode
